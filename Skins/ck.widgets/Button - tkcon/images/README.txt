How to make button:
1. Open original image in paint.net
2. Resize image: Menu->Image->Resize...
3. Resize canvas: Menu->Image->Canvas Size...
a) Enter the "width" as original width * 3
b) Choose "Anchor" as "Left"
4. Copy the original image 2 times to left side in different layers
a) Make the second image moved by 1 pixel right-down to simulate the click
5. Apply b&w filter to the first image: Menu->Adjustments->Black and White
6. Save the result image in PNG format
